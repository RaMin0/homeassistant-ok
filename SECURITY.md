# Security Policy

## Supported Versions

Only the latest published release is actively supported for security fixes. Older releases are
best-effort and may be asked to upgrade before a fix can be verified.

## Reporting A Vulnerability

Report security issues privately with GitHub's private vulnerability reporting from the
repository **Security** tab. Do not open a public issue for security reports.

Do not include OK account credentials, login tokens, Home Assistant storage files, logs, or
unsanitized API captures in reports. Diagnostics generated by this integration redact known
OK app, device, and token fields.
